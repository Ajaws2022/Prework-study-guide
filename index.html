HTML

  • The head contains critical information for rendering the site
  • The information contained can affect our seo, language, and content interpretation
  • Another important factor in the head is the data for how the site will scale across devices

CSS

  • Allows us to add style to our html elements
  • 3 types are Inline, Internal, and External
  • External sheets are preffered as they provide a link to the css rules rather than a direct embed
  • Wildcard symbols such as "*" can be used to apply rules to the entire set of elements
  • External .css rules will need to be linked to the html document in order to be visible
  • Class Selectors are used to designate property assignments to particular elements
  • CSS Rule: selector{property:value; property:value;}

  • git add: allows the system to automatically track and stage commits
