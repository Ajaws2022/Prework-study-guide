HTML

  • The head contains critical information for rendering the site
  • The information contained can affect our seo, language, and content interpretation
  • Another important factor in the head is the data for how the site will scale across devices

CSS

  • Allows us to add style to our html elements
  • 3 types are Inline, Internal, and External
  • External sheets are preffered as they provide a link to the css rules rather than a direct embed
  • Wildcard symbols such as "*" can be used to apply rules to the entire set of elements
  • External .css rules will need to be linked to the html document in order to be visible
  • Class Selectors are used to designate property assignments to particular elements
  • CSS Rule: selector{property:value; property:value;}

GIT

  • git status will allows us to check our git connection
  • git checkout: allows for navigation between branches
  • git checkout -b: creates a new branch and moves us onto it
  • git add: allows the system to automatically track and stage commits

JavaScript

  • Introduces decision making and interactivity to web pages
  • This is done in script.js documents and uses the Control Flow of Top to Bottom
  • JavaScript is case sensitive, this is important to remember when creating variables and functions
  • JS uses a variety of tools to accomplish its intuitive nature, these include but are not limited to Variables, let declarations, const. declarations, if and else-if statements, for loops, and functions
  • Variables are named containers that contain data, to be used they must first be declared with the let keyword and then given a value that it represents
  • Variables can contain strings(a sequence of text placed in quotes) Numbers(literal), Boolean (True or False), Arrays (multiple values contained in one variable), and Objects (anything in the scope of JavaScript as everything in JS is dealing with objects)
  • Array variables are iterated over using a zero-index system which labels the first item in the array as [0} then [1] and so on for the length of the array
  • For loops are a handy way to manage how the system iterates an array. Rather than manually typing the same condition for each item in an array the code is simply looped to go through the list at its given starting point
  • Operators are mathematical symbols used to combine multiple variables for a new value
  • Functions are a set of instructions for the computer to perform a specific task labelled as a key word
  • Functions can contain if statements and for loops
